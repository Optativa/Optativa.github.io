Setup

Setting up your wiki dev environment

Initial setup

  1. If you don't have it install npm with your preferred package manager
  2. Install gitbook by running npm install -g gitbook-cli
  3. (Optional) npm might want to write in /usr/local/lib obviously don't do this, create a folder for npm in your home called .npm_global and then call npm config set prefix '~/.npm_global'
  4. Clone the dev repository located at https://github.com/Optativa/optawiki_dev
  5. Also clone the github page repository https://github.com/Optativa/Optativa.github.io

Congrats you should be set up to modify these pages

Modifying your first page

If you inspect optatiwiki_dev you should see a bunch of readmes, these are the ones that make up the wiki's pages, you can modify them as you please.

If you want to preview your changes you need to run gitbook build followed by gitbook serve, this will let you preview the wiki locally.

Once you have made all the changes you wanted move to the deploy step

Deploying

  1. First make sure you have commited all your changes to optawiki_dev
  2. Run gitbook build to get the latest build of the wiki
  3. Make sure you haven't broken anything :)
  4. Run the deploy script that should be inside optawiki_dev called deploy.sh
  5. Follow the deploy script steps

Congrats your page should be uploaded and visible now.
